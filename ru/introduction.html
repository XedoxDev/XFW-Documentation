Назад к меню

Быстрое введение для Фреймворка XedoxFramework.

Введение

Скачайте последнюю версию библиотеки, и поместите файл по дирректории /app/libs/

В файл /app/build.gradle, в блок dependencies добавьте файл в зависимости приложения:

dependencies {
  implementation files("libs/XFW.aar")
}

Далее, в активности создайте объект класса XSurfaceView, передав туда контекст, а также стартовый экран (это будет чуть позже). Пример:

@Override
public void onCreate(Bundle bundle) {
  super(bundle);
  XSurfaceView xview = new XSurfaceView(this, new MainScreen());
  setContentView(xview);
}

Создайте наследник класса ScreenAdapter, это и есть тот самый экран, но обычно такие вещи называют сценами, переопределяем основные методы start() и render(Canvas). Пример:

public class MainScreen extends ScreenAdapter{
  public void start() {
  }
  
  public void render() {
  }
}
