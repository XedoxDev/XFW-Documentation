Назад к меню

Быстрое введение для Фреймворка XedoxFramework.

RenderManager

RenderManager - это специальный класс, для отрисовки различных фигур.

Для начало, нужно создать обьект этого класса:

private RenderManager rm;
    
public void start() {
  rm = new RenderManager(this);
}

Теперь, в render, можно рисовать. В RenderManager есть такие методы как rect, а стилем для него будет являтся сам RenderManager, так как он является наследником класса Paint.

public void render(Canvas canvas) {
  rm.setColor(0xFF00FF00); //GREEN
  rm.rect(100, 100, 100, 100);
}

Написание схоже с синтаксисом языка Processing. Так что много чего будет схоже.

Класс Object

Это не тот стандартный класс, о котором вы могли подумать, он находится в пакете com.xdxfw.graphics. Он является абстрактным классом, в также наследником Paint, чтобы стиль наследника можно было менять напрямую у обьекта.

В Фреймворке, есть несколько стандартных классов-обьектов, мы разберем только Line и Shape

Обьекты всех наследников Object можно добавлять в RenderManager, и потом в render(Canvas) вызывать у его метод displayObjects() и все добавленве обьекты отрисуются, там очень гибко сделана система добавления, удаления и получения обьектов, так что разбираться будет очень просто.

Приведу небольшой пример с обьектами Line:

public class MainScreen extends ScreenAdapter {
  RenderManager rm;
  
  public void start() {
    rm = new RenderManager(this);
    Line l1 = new Line(0, 0, 100, 100);
    Line l2 = new Line();
    l2.setPoint1(200, 50);
    l2.setPoint2(300, 200);
    rm.addObject(l1);
    rm.addObject(l2);
  }
  
  public void render() {
  }
}
